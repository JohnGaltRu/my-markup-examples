Free text showing the possibility of writing text in html in various forms

Tags for HTML text can be divided into the following:

  • Header tags: <h1....h6>;
  • Paragraphs, text wrappers
  • Tags for text formatting;
  • Tags for entering "computer" text
  • Tags for quotes and definitions

Paragraphs, text wrappers

First of all, I will point out that the <p> tag is used to create paragraphs.

You can write text without any action from the browser using the <span> tag. Using the span tag, you can select some of the information inside other tags and set your own style for it.

For example, inside a paragraph you can change the color and size of the first letter or word.

The <br> tag is used to wrap a line.

Tags for entering "computer" text

The most used tags here are <code> and <pre>

The <code> tag is used to highlight code fragments:
function alertName (name) {
alert(`Hello ${name}!`)
}

The <pre> tag allows you to display text on the screen, retaining the original formatting. Spaces and line breaks are not deleted:

Tags for quotes and definitions

The most used tags here are <abbr> and <blockquote>

The <abbr> tag is used to format abbreviations: 'BEM is a component-based approach to web development'.

The <blockquote> tag highlights quotes within the document, highlighting it with indents and line breaks.
